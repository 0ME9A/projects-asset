## Project Setup

To get started with the "Cards" project, follow these steps:

1. Create a new folder on your computer to store the project files. You can name it **Cards** or any other preferred name.

2. Inside the "Cards" folder, create two new files: `index.html`, `style/style.scss` and `img/`. You can use a simple text editor or a code editor like [Visual Studio Code](https://code.visualstudio.com/Download) to create these files.

## Compiling SCSS to CSS

To use the SCSS styles in your project, you need to compile the `style.scss` file into a regular CSS file. If you don't have a build process set up, you can use online tools or install a package like `node-sass` to compile SCSS to CSS or if you are using VS-Code install Live Sass Compiler.

1. Save all the changes made to the `index.html` and `style.scss` files.

2. Open the `index.html` file in your web browser to view the "Cards" project.

Congratulations! You have successfully set up the "Cards" project and created a responsive grid of card elements.
