## Best Practices with Zustand

### Lightweight Global State
- **What to Do**: Use Zustand for simple, performant state management.
- **Example**:
  ```javascript
  import create from "zustand";

  const useStore = create((set) => ({
    count: 0,
    increment: () => set((state) => ({ count: state.count + 1 })),
    reset: () => set({ count: 0 }),
  }));

  function Counter() {
    const { count, increment, reset } = useStore();
    return (
      <div>
        <p>Count: {count}</p>
        <button onClick={increment}>Increment</button>
        <button onClick={reset}>Reset</button>
      </div>
    );
  }
  ```
- **Benefit**: Minimal boilerplate, ideal for small to medium apps.
- **Link**: See [Zustand Docs](https://docs.pmnd.rs/zustand/getting-started/introduction).

### Optimizing with Selectors
- **What to Do**: Use selectors to prevent unnecessary re-renders.
- **Example**:
  ```javascript
  const useCountSelector = useStore((state) => state.count);
  function DisplayCount() {
    const count = useCountSelector();
    return <p>Selected Count: {count}</p>;
  }
  ```
- **Benefit**: Improves performance by isolating state slices.

## Best Practices with Recoil

### Atomic State Management
- **What to Do**: Define atoms for granular state control.
- **Example**:
  ```javascript
  import { atom, useRecoilState } from "recoil";

  const countState = atom({
    key: "countState",
    default: 0,
  });

  function CounterRecoil() {
    const [count, setCount] = useRecoilState(countState);
    return (
      <div>
        <p>Count: {count}</p>
        <button onClick={() => setCount(count + 1)}>Increment</button>
      </div>
    );
  }
  ```
- **Benefit**: Fine-tuned state updates for complex UIs.
- **Link**: See [Recoil Docs](https://recoiljs.org/docs/introduction/installation).

### Syncing with Async Data
- **What to Do**: Manage async state with selectors.
- **Example**:
  ```javascript
  const userDataState = selector({
    key: "userDataState",
    get: async () => {
      const response = await fetch("https://api.example.com/user");
      return response.json();
    },
  });

  function UserDisplay() {
    const userData = useRecoilValue(userDataState);
    return <p>User: {userData.name}</p>;
  }
  ```
- **Benefit**: Handles dynamic data efficiently.
